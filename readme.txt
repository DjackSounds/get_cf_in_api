=== Get CF in API ===
Tags: custom fields, REST API, get, ACF, advanced custom fields
Requires at least: 1
Tested up to: 5.3.2
License: GPLv2

add the possibility to get all your custom fields (including ACF) with the Wordpress REST API

https://github.com/DjackSounds/get_cf_in_api

== Installation ==
Upload « get-cf-in-api » folder to the « /wp-content/plugins/ » directory.
Activate the plugin through the « Plugins » menu in WordPress.
